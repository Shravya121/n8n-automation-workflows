## Setup Instructions

### Prerequisites
- n8n (self-hosted or cloud)
- Groq API key (free at console.groq.com)
- Google account

### How to Use
1. Download the workflow JSON file
2. Import into n8n
3. Replace YOUR_GROQ_API_KEY with your actual key
4. Replace YOUR_SHEET_ID with your Google Sheet ID
5. Connect your Google/Gmail credentials
6. Activate workflow
